Text

Download the latest version of Atlas Tool from mdsitton's GitHub and extract it.

Create a new folder inside Atlas Tool's folder with a suitable name (such as 'PTB').

Open the Clone Hero Launcher, open the 'Settings' tab, and locate 'CloneHero Install Directory'. Click on the folder icon to the right, then 'Yes'.

Create a copy of the 'Clone Hero_Data' folder so you have an easily accessible backup incase anything goes wrong. Once you have done that, copy the folder to the 'PTB' folder we created earlier.

Once that's finished copying open Command Prompt and 'cd' to Atlas Tool's folder. (If you put the folder on your Desktop you'd type `cd Desktop\atlastool`.

In Command Prompt type `atlastool.exe -x -i "PTB\Clone Hero_Data"` and wait. If it says 'Loading assets from PTB\Clone Hero_Data' then you did it right. If it says anything else then make sure you entered the command correctly.

Once the next message appears, it's finished extracting.

Atlas Tool will have created a folder named after the version number. (The latest PTB at the time of writing is 'v1.0.0.3493-master' so that's what the folder is called. The textures are stored in a folder called 'fiveFretAtlas' located within that folder. Change whatever textures you want to change but DO NOT change the file names.

Once you have finished changing the textures go back into Command Prompt and type `atlastool.exe -c -i .\FOLDER NAME` while replacing 'FOLDER NAME' with the name of the folder that Atlas Tool created earlier.

Once the process has complete, navigate back to the version folder and there should now be two image files.

Download the latest Windows version of UABEA from their GitHub (you may need to scroll down a bit) and extract it.

Open `UABEAvalonia.exe` (scroll to the bottom of the folder). Select `File` and then `Open`. Navigate back to your 'CloneHero Install Directory', open the 'Clone Hero_Data' folder, and open `globalgamemanagers`.

If you get the popup shown below, scroll to the bottom, click on the option at the bottom, and click 'Ok'.

Click on 'Name' at the top until the triangle is pointing pointy side up, and scroll down until you file the file beginning with 'sactx'.

Click on 'Plugins' in the right side pannel, then 'Edit texture', and 'Ok'.

Now click on 'Load' and navigate to Atlas Tool's folder. Open the version number's folder and double click on the image file ending in '-changed' (the second one if you're sorting alphabetically).

Click on 'Save' and wait until the window closes. The program may stop responding, THIS IS NORMAL. Just keep waiting. Once the window closes, select 'File' in the top bar, then 'Save'. Navigate to an easily accessible folder (such as your desktop) and click 'Save'. Once you have done that you can close UABEA.

Go back into your CloneHero Install Directory and open the 'Clone Hero_Data' folder (NOT the copy we made earlier). Copy and paste the 'resources.assets' file that UABEA created into your 'Clone Hero_Data' folder and select 'Replace the file in the destination'.

You have now modded your game. Congratulations!
